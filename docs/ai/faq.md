# Frequently Asked Questions (FAQ)

## Getting Started

### How do I start using ZZP HUB?
1. Create an account and log in
2. Go to Settings → Company Profile (Bedrijfsprofiel)
3. Fill in all required company information
4. Add your first client in "Relaties"
5. Create your first invoice

### What information do I need?
- Company name and address
- KVK number
- VAT number (BTW-nummer)
- IBAN and bank name
- Payment terms

## Invoices

### How do I send an invoice?
1. Open the invoice
2. Click "Acties"
3. Choose "Verstuur via e-mail" or "Download PDF"

### Can I edit sent invoices?
No, sent invoices cannot be edited. Create a new one if needed.

### How do I convert a quotation?
Change status to GEACCEPTEERD and click convert.

## VAT

### Which VAT rate should I use?
- 21%: Most services (default)
- 9%: Specific categories (consult advisor)
- 0%: Exports/special cases

### How do I file VAT returns?
Use the BTW-aangifte overview, then file via Mijn Belastingdienst.
